Muted Ink / VS Code

What the settings do

Subtraction, mostly

Minimap, breadcrumbs, command center, layout control, and per-editor action buttons are all off — each duplicated something the editor already showed.

Activity bar on top

It costs a row instead of a column, which is the cheaper axis on a widescreen display.

A real font stack

Ghostty bundles JetBrains Mono. VS Code bundles nothing, and fc-match answers Noto Sans — proportional — when it’s missing. The stack falls back through Source Code Pro to monospace.

formatOnSave stays off

Formatting is a per-project decision. A global default rewrites files in repos that never asked for it.

Bracket pairs, no guides

Colorized pairs in teal, blue, and purple carry the nesting; the vertical bracket guides that would restate it are off.

One accent, held back

Teal appears on the cursor, the active line number, the active tab rule, the focus ring, and buttons. Nothing else competes for it.

./install.sh vscode  ·  the theme is registered from a generated .vsix, then symlinked back into the repo, so editing it here changes the live config on the next window.
